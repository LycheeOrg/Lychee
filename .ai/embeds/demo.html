Quick Start

Add the widget to your website in 3 simple steps:

1. Include the widget files

<link rel="stylesheet" href="https://your-lychee.com/embed/lychee-embed.css"> <script src="https://your-lychee.com/embed/lychee-embed.js"></script>

2. Add the embed container

<div data-lychee-embed data-api-url="https://your-lychee.com" data-album-id="your-album-id" data-layout="justified" data-show-title="true" data-show-exif="true" ></div>

3. The widget auto-initializes!

That's it! The widget will automatically detect elements with data-lychee-embed and initialize them.

Configuration Options

Customize the widget behavior with data attributes:

Attribute Type Default Description
data-api-url String Required Base URL of your Lychee instance
data-album-id String Required ID of the album to embed
data-layout String "justified" Layout type: square, masonry, grid, justified, filmstrip
data-spacing Number 8 Gap between photos in pixels
data-target-row-height Number 320 Target height for justified layout rows
data-target-column-width Number 300 Target width for grid/masonry columns
data-show-title Boolean true Show album title
data-show-description Boolean true Show album description
data-show-captions Boolean true Show photo captions in lightbox
data-show-exif Boolean true Show EXIF data in lightbox
data-theme String "light" Theme: "light" or "dark"

Programmatic API

For advanced use cases, you can initialize the widget programmatically:

import { createLycheeEmbed } from './lychee-embed.js'; // Initialize widget const app = createLycheeEmbed('#my-container', { apiUrl: 'https://your-lychee.com', albumId: 'your-album-id', layout: 'justified', spacing: 12, targetRowHeight: 350, showTitle: true, showExif: true, theme: 'dark' });
